Library, bookshelf, and obtaining content

In this section we identify the requirements on how to obtain content from various sources and how it is presented in the library view, sometimes called the bookshelf view. The search and discovery of materials is an important part of these requirements. It also covers the organization of the content by the end user.

  • The application must be able to obtain content from a variety of sources. A variety of library formats must be supported, e.g. OPDS,DOPD.

Navigation inside book

In this section we define the navigation a user can perform. It is limited to the book itself.

The user must be able to determine their location in the content (Where am I?).
nav-310 : Read navigation information
Screen reader users must be able to navigate through content by headings, block items, lines, words and characters.
